# e-market-app
 e-market-app is small version of amzon where yo can buy items from it .
 This application was designed using flutter as interfaces and using [nodejs](https://github.com/allaithsaad/nodeJs-api-for-e-market) as a server to run the system. MongoDB was used as a database and cloudinary was used to upload images to it.
